# Release notes
Release notes for scolorpicker.

## v1.1.0 | 13/06/2023
**Added**
- Color code output in different modes (RGB, HSV, HSL, etc.)
    Available modes:
    - `hex` - Hexadecimal
    - `rgb` - Red, Green, Blue
- Ability to choose to output the color code to the terminal instead of the clipboard
- Amazing new CLI!
